ReadMe
Install Python: Make sure Python is installed on your system. You can download it from python.org.
Clone the Repository:

git clone https://github.com/Stella-28/ChatBOT.git


Install Dependencies:


pip install -r requirements.txt
Extract the Corpus:
Run the script to extract text from the PDF and save it to corpus.txt:
python extract_corpus.py
Run the Application:

python app.py


Access the Chatbot: Open your web browser and go to http://127.0.0.1:5000/.
